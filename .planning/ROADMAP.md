# Roadmap: API-Pulse

## Milestone 1: Minimum Viable Engine

### Phase 1: The Scout (Scraper & Analyzer)
- **Goal**: Develop the Python/Playwright scraper to identify 50+ public APIs and use Gemini to extract JSON metadata.
- **Status**: Not Started

### Phase 2: The Heartbeat (Validator)
- **Goal**: Implement an async Python checker to verify endpoint statuses and update `data.json`.
- **Status**: Not Started

### Phase 3: The Dashboard (Frontend)
- **Goal**: Build a React/Vite/Tailwind static site with search/filters and dark-mode UI.
- **Status**: Not Started

### Phase 4: CI/CD & Review
- **Goal**: Configure GitHub Actions for deployment and PR review.
- **Status**: Not Started
